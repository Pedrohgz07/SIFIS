Acerca de SIFIS

Sistema de Identificación Forense de Imágenes Sintéticas generadas por IA mediante Algoritmos de Aprendizaje.

El proyecto

¿Qué es SIFIS?

SIFIS es un sistema web de identificación forense desarrollado para determinar si una imagen digital fue generada por Inteligencia Artificial. Ante el avance acelerado de los modelos generativos como GAN y algoritmos de difusión, que producen imágenes con un realismo difícil de distinguir a simple vista, se hace necesaria una herramienta accesible que apoye la verificación de contenido digital en entornos educativos e institucionales.

El sistema implementa una Red Neuronal Convolucional (CNN) basada en Transfer Learning con EfficientNet-B0 para la clasificación binaria de imágenes reales y generadas por IA, entrenada sobre el dataset 140K Real and Fake Faces de Kaggle. La evaluación del modelo emplea métricas como Accuracy, AUC, ROC Curve, Precision, Recall y F1-Score.

Tecnologías utilizadas

Stack tecnológico seleccionado para el entrenamiento, evaluación y despliegue del modelo de detección.

Python 3 Lenguaje principal para el desarrollo, entrenamiento y evaluación del modelo
TensorFlow 2.x Framework de aprendizaje profundo para construir, entrenar y desplegar la red neuronal
Keras API de alto nivel integrada en TensorFlow para diseñar la arquitectura del modelo
EfficientNet-B0 CNN con Transfer Learning (ImageNet) — extractor de características con bajo costo computacional
Transfer Learning Reutilización de pesos preentrenados en ImageNet para reducir tiempo de entrenamiento
Fine-Tuning Descongelamiento de las últimas 30 capas para especializar la red en detección de IA
Data Augmentation RandomFlip, RandomRotation y RandomZoom para reducir sobreajuste (overfitting)
Adam Optimizer Optimizador con dos fases de aprendizaje: lr 0.001 → 0.00001 en fine-tuning
scikit-learn Métricas de evaluación: Accuracy, AUC, ROC Curve, Precision, Recall, F1-Score
NumPy Manipulación de arreglos y operaciones matemáticas sobre imágenes
Matplotlib Visualización de curvas de entrenamiento, ROC y matriz de confusión
140K Real & Fake Faces Dataset de Kaggle con 140,000 rostros reales y generados por IA para entrenamiento
TensorFlow Dataset API Carga eficiente por lotes (batch) con prefetch automático para optimizar GPU
GPU NVIDIA (Kaggle) Aceleración CUDA para el entrenamiento — TensorFlow detecta la GPU automáticamente
